# 📊 E-Commerce Purchases Data Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an **E-Commerce Purchases** dataset using Python. The goal is to explore customer purchasing behavior, identify spending patterns, and answer business-related questions through data analysis and visualization.

The project demonstrates the complete data analysis workflow, including data cleaning, statistical analysis, and visualization using industry-standard Python libraries.

---

# 🎯 Objectives

- Explore the dataset structure
- Perform data cleaning and preprocessing
- Analyze customer purchase behavior
- Identify spending trends
- Answer business questions using data
- Create meaningful visualizations
- Generate actionable business insights
